# 🌐 Domain-Agnostic Universal Agent-Led R&D Loop Guide

본 프레임워크는 사용자가 파이썬 스크립트를 직접 실행하는 것이 아니라, **AI 에이전트(Antigravity)가 `schedule` 도구를 활용해 60초마다 스스로 깨어나 자율 R&D 루프를 실행하도록 조종하는 에이전트 주도형 템플릿**입니다.

새로운 프로젝트에 이 템플릿을 복사한 후, 에이전트에게 60초 루프 구동을 지시하면 에이전트가 스스로 스케줄링을 걸고 무인 튜닝을 수행합니다.

---

## 🛠️ 새 프로젝트 적용 시 작성해야 하는 4대 필수 정의 요소

새 프로젝트 루트에 이 템플릿을 복사한 후, `config/project_rules.json` 및 `.agents/AGENTS.md`에 아래 4가지를 **반드시 정의하여 작성해야 합니다**:

1. **목표 평가 지표 (Objective Metric Defined)**: 최적화하려는 성능 지표를 정의해야 함
2. **데이터 누수 & 과적합 하드 블록 상한선 (Overfitting Cap Defined)**: 즉시 탈락시킬 착시 임계치를 정의해야 함
3. **현실 환경 마찰 모델 (Environment Friction Defined)**: 측정 오차, 수수료, 데이터 노이즈 등 마찰 모델을 정의해야 함
4. **Out-of-Sample (OOS) 검증 기준 (OOS Decay Target Defined)**: 미공개 OOS 검증 구간의 최소 성과 보존율을 정의해야 함

---

## 🤖 AI 에이전트에게 60초 자율 루프 구동 지시하는 방법 (Prompting Guide)

이 폴더를 새 프로젝트에 복사한 뒤, AI 에이전트에게 다음과 같이 채팅창에 입력하십시오:

> **"이 폴더에 있는 템플릿을 기반으로 60초 자율 R&D 스케줄러 루프를 돌려줘. 매 60초마다 `schedule` 도구를 사용해 대기하고, 깨어나면 `python auto_rnd_loop.py`를 직접 1회씩 실행하며 최적의 모델 가설을 지속적으로 탐색해줘."**

---

## 🔁 에이전트 자율 60초 루프 작동 매커니즘 (Agent Self-Loop Mechanism)

1. **가설 적용**: 에이전트가 소스 코드에 새로운 모델 아키텍처/하이퍼파라미터 가설을 반영합니다.
2. **코드 직접 실행**: 에이전트가 `run_command` 도구로 `python auto_rnd_loop.py`를 1회 실행하여 테스트합니다.
3. **과적합 검증 및 챔피언 승격**: 실행 결과를 분석하여 과적합 방화벽(`leakage_detector.py`)을 통과하고 성능이 향상되면 챔피언 모델로 승격시킵니다.
4. **60초 자율 대기**: 에이전트가 스스로 `schedule(DurationSeconds=60)` 도구를 호출하여 타이머를 세팅하고 잠에 듭니다 (turn 종료).
5. **무한 순환**: 60초 후 시스템에 의해 에이전트가 깨어나면 다시 **1단계**부터 루프를 반복 구동합니다.
