# 🤖 Reusable Agent-Led Autonomous AI R&D Loop Suite

이 저장소는 AI 에이전트(Antigravity)가 `schedule` 도구를 사용하여 스스로 60초마다 깨어나 자율 R&D 가설 실험, 백테스트, 과적합 검증을 무인 상태에서 지속 수행하도록 조종하는 **에이전트 주도형 60초 자율 루프** 표준 템플릿 패키지 모음입니다.

사용자가 파이썬 코드를 로컬에서 수동으로 실행할 필요 없이, 프로젝트에 이 템플릿을 복사한 뒤 에이전트에게 60초 루프 작동을 지시하면 에이전트가 알아서 순환 동작을 반복합니다.

---

## 📂 저장소 패키지 구성 (Directory Layout)

### 1. 🌐 [universal_ai_framework/](./universal_ai_framework/)
- **특징**: 주식/금융에 종속되지 않은 **완전 도메인 독립형 범용 AI R&D 템플릿**.
- **적용 도메인**: 수요 예측, 센서 예측, 비즈니스 지표 등 일반 머신러닝/딥러닝 튜닝.
- **작동 방식**: 에이전트가 60초마다 `schedule`을 걸고 깨어나 `auto_rnd_loop.py`를 직접 1회씩 호출하며 가설 탐색 및 챔피언 동기화.

### 2. 📈 [financial_quant_framework/](./financial_quant_framework/)
- **특징**: 주식/암호화폐/금융 시계열 R&D에 특화된 **금융 퀀트 전용 AI R&D 템플릿**.
- **적용 도메인**: 계량 투자 포트폴리오 최적화, 변동성 제어, 트레이딩 알파 마이닝.
- **작동 방식**: 에이전트가 60초마다 `schedule`을 걸고 깨어나 `auto_tuner_loop.py`를 직접 1회씩 호출하며 백테스트 및 5대 과적합 검증(DSR, PBO, MDD Zero, Slippage 20 bps) 수행.

## ⚡ 빠른 적용 가이드 (Quick Start)

1. 이 저장소를 로컬로 복사하거나 필요한 프레임워크 폴더를 새 프로젝트 루트 폴더에 복사합니다.
2. 복사한 폴더 내부의 `README.md` 가이드라인에 맞추어 필수 규칙 및 임계치 설정을 작성합니다.
3. AI 에이전트에게 다음과 같이 지시하여 60초 자율 루프를 시작하게 만듭니다:
   > **"이 템플릿 폴더를 기반으로 60초 자율 R&D 스케줄러 루프를 돌려줘. 매 60초마다 schedule 도구를 사용해 대기하고, 깨어나면 코드를 직접 1회씩 실행하며 최적의 가설을 지속적으로 탐색해줘."**
